# Critérios de Avaliação Interna — ExtraUFLA

## Definition of Done (DoD)

Um item do backlog é considerado concluído quando:

- [ ] Está documentado no repositório
- [ ] Foi revisado por pelo menos um outro integrante
- [ ] Atende aos critérios de aceitação definidos
- [ ] Está versionado no GitHub (commit na branch correspondente)
- [ ] Está alinhado com o objetivo da sprint e da entrega da disciplina

## Critérios de autoavaliação por sprint

| Critério | Peso |
|---|---|
| Artefatos entregues conforme planejado | Alto |
| Qualidade e clareza da documentação | Alto |
| Participação de todos os integrantes | Médio |
| Uso correto do Scrum (backlog, sprint, revisão) | Alto |
| Commits rastreáveis por integrante | Médio |
